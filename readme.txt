GitHub Pages Visitekaartje Site

1) Upload alle bestanden in deze map naar een nieuwe GitHub repository.
2) Ga naar Settings → Pages → kies main branch, root folder → Save.
3) Je site staat nu live op https://<user>.github.io/<reponaam>/
4) Om te editen: open config.json in GitHub, klik op het potloodje, pas tekst/links aan en commit.
5) Refresh je site: klaar!

Voor een eigen domein: voeg bij Settings → Pages je domein toe, en maak een CNAME record in je DNS naar <user>.github.io
